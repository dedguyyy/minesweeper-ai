# Minesweeper — Game and Reasoning AI

Implements the Minesweeper game logic and a simple AI that reasons about
safe and mine cells using logical sentences.

**Overview**
- Contains `Minesweeper` game model, `MinesweeperAI` with knowledge-based reasoning.

**Quick Start / Usage**
Open a Python REPL or script and import the classes to experiment with the AI.

```python
from minesweeper import Minesweeper, MinesweeperAI
game = Minesweeper()
ai = MinesweeperAI()
```

**Requirements**
- Python 3.6+
